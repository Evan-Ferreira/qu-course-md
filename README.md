# qu-course-md

A personal archive of Queen's University course material—lectures and required readings—stored as markdown for easier access and review.

## Purpose

This repository keeps course content in a clean, searchable, version-controlled format. Plain text is easier to navigate than scattered PDFs on an LMS, and it works well with modern study workflows that benefit from having full context in one place.

Each course has its own directory containing lecture notes, assigned readings, and related material for the semester.

## Structure

```
enin-240/
  lecture-1.md
  lecture-2.md
  ...
  some-required-reading.md
```

Additional courses may be added over time as needed.

## Usage

Treat this like personal course notes: read, review, and use it to clarify concepts you've already engaged with in class. Having material in text form is a reference aid—not a substitute for doing the work yourself.
